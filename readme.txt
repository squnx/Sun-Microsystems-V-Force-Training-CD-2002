V-Bots CD-ROM

PC, Mac & Unix Version


RUNNING THE CD
--------------
Double-click the file called "index.htm". If the file does not open drag and drop into your open browser.


SYSTEM REQUIREMENTS
-------------------
This CD works with Netscape Navigator 4.5 and above and Internet Explorer 4.0 and above. If you don't have a browser, please click on the Install folder and select a browser compatible to your system (e.g. Mac, PC, or Unix). 

For best results, use Internet Explorer version 5.0, and a machine with at least 64MB RAM, color monitor with 800 x 600 or higher resolution, capable of showing 16 bit (Hi Color setting).
